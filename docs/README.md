## 注文管理システム

## 概要
このアプリは、学校で行われた文化祭で使用していました。
まだ初心者の頃に作っていたので、ディレクトリ構成や、バックエンドの実装がひどいですが、ご容赦ください。

## 必須環境
- Docker Engine

## 初期設定
プロジェクトのルートディレクトリに`.env`ファイルを追加して、下記の内容を入力してください
```.env
ROOT_PASSWORD=<強力なパスワード>
```
`<強力なパスワード>`には`""`を入力しなくても大丈夫です

## 起動
### 起動時
```
docker compose up -d
```

### 終了時
```
docker compose down
```
